依赖管理

基于spring-boot-starter-parent，包含了常用的各种库的依赖管理和插件管理

工具库

基于Apache Commons、jasypt、twelvemonkeys、poi-tl等工具库，包含了基础、文件压缩、加解密、地理信息、图像、IO、PDF、OFFICE、Jakarta参数校验用途的工具类

框架

基于Spring Framework和Mybatis Plus，提供了例如：MongoDB CRUD辅助、Redis CRUD辅助、Mybatis Plus CRUD辅助、业务异常体系、第三方接口请求辅助、基础Web过滤器、基础Web拦截器、Web工具类等内容

Spring Boot Starter

基于Spring Boot Starter 和 Pangju Framework，提供了例如：动态Redis数据源、动态MongoDB数据源、Mybatis Plus插件管理、测试、Json字段脱敏、Json字段加解密、接口参数加解密、接口签名、接口限流、接口幂等性、接口日志、接口响应包装、全局异常处理等内容
