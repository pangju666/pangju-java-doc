# 枚举

## 水印方向
`io.github.pangju666.commons.image.enums.WatermarkDirection`

| 枚举值          |       说明       |                          
|--------------|:--------------:|
| TOP_LEFT     |  左上角，紧贴上边与左边   |
| TOP          | 顶部居中，紧贴上边并水平居中 |
| TOP_RIGHT    |  右上角，紧贴上边与右边   |
| RIGHT        | 右侧居中，紧贴右边并垂直居中 |
| CENTER       |  中心位置，水平与垂直居中  |
| LEFT         | 左侧居中，紧贴左边并垂直居中 |
| BOTTOM       | 底部居中，紧贴下边并水平居中 |
| BOTTOM_RIGHT |  右下角，紧贴下边与右边   |
| BOTTOM_LEFT  |  左下角，紧贴下边与左边   |
