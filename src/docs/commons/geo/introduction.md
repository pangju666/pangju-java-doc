# 地理信息模块

## 概述
geo 模块主要用于地理信息相关项目的开发，提供了一系列功能，例如：
- 坐标类型
- 坐标数据结构
- GEO相关常量
- 坐标工具类

## 安装
```xml-vue
<dependency>
    <dependency>
        <groupId>io.github.pangju666</groupId>
        <artifactId>pangju-commons-geo</artifactId>
        <version>{{ commonsVersion }}</version>
    </dependency>
</dependency>
```
