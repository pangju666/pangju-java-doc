# 异常处理

## 概述
这个模块我集成了`spring-boot-starter-web`和[`Pangju Framework Web`](/framework/web/introduction)。

## 依赖

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>

<dependency>
    <groupId>io.github.pangju666.framework</groupId>
    <artifactId>framework-web</artifactId>
</dependency>
```

## 引入
```xml
<dependency>
    <dependency>
        <groupId>io.github.pangju666.framework.boot</groupId>
		<artifactId>framework-spring-boot-starter-web</artifactId>
    </dependency>
</dependency>
```
