# Web 加密

## 概述
这个模块我集成了[`Pangju Starter Web`](/starter/web/introduction)和[`Pangju Starter Json Crypto`](/starter/json/crypto/introduction)。

## 依赖

```xml
<dependency>
    <groupId>io.github.pangju666.framework.boot</groupId>
    <artifactId>framework-starter-web</artifactId>
</dependency>
<dependency>
    <groupId>io.github.pangju666.framework.boot</groupId>
    <artifactId>framework-starter-json-crypto</artifactId>
</dependency>
```

## 引入
```xml
<dependency>
    <dependency>
        <groupId>io.github.pangju666.framework.boot</groupId>
		<artifactId>framework-starter-web-crypto</artifactId>
    </dependency>
</dependency>
```
