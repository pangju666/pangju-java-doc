# Web 日志

## 概述
这个模块我集成了[`Pangju Starter Web`](/starter/web/introduction)、`commons-collections4`和`disruptor`。

## 依赖

```xml
<dependency>
    <groupId>io.github.pangju666.framework.boot</groupId>
    <artifactId>framework-spring-boot-starter-web</artifactId>
</dependency>

<dependency>
    <groupId>org.apache.commons</groupId>
    <artifactId>commons-collections4</artifactId>
</dependency>
<dependency>
    <groupId>com.lmax</groupId>
    <artifactId>disruptor</artifactId>
</dependency>
```

## 引入
```xml
<dependency>
    <dependency>
        <groupId>io.github.pangju666.framework.boot</groupId>
		<artifactId>framework-spring-boot-starter-web-log</artifactId>
    </dependency>
</dependency>
```
