# Spring 模块

## 概述
这个模块我集成了`spring-boot-starter`和[`Pangju Framework Spring`](/framework/spring/introduction)。

## 依赖

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter</artifactId>
</dependency>

<dependency>
    <groupId>io.github.pangju666.framework</groupId>
    <artifactId>framework-spring</artifactId>
</dependency>
```

## 引入
```xml
<dependency>
    <dependency>
        <groupId>io.github.pangju666.framework.boot</groupId>
		<artifactId>framework-starter-spring</artifactId>
    </dependency>
</dependency>
```
