# 校验 模块

## 概述
这个模块我集成了`spring-boot-starter-validation`和[`Pangju Commons Validation`](/commons/validation/introduction)。

## 依赖

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-validation</artifactId>
</dependency>

<dependency>
    <groupId>io.github.pangju666.commons</groupId>
    <artifactId>commons-validation</artifactId>
</dependency>
```

## 引入
```xml
<dependency>
    <dependency>
        <groupId>io.github.pangju666.framework.boot</groupId>
		<artifactId>framework-spring-boot-starter-validation</artifactId>
    </dependency>
</dependency>
```
